# 📚 StudyNow – AI Study Focus & Distraction Control App

![Android](https://img.shields.io/badge/Platform-Android-green)
![Language](https://img.shields.io/badge/Language-Java-blue)
![IDE](https://img.shields.io/badge/IDE-Android%20Studio-orange)
![Status](https://img.shields.io/badge/Status-Active-success)

StudyNow is an **Android productivity application** designed to help students stay focused during study sessions.  
The app allows users to **set study goals, run focus sessions, track productivity, and access AI-based study assistance**.

The goal of this project is to **reduce distractions and improve study efficiency using structured focus sessions and intelligent support tools.**

---

# 🚀 Features

### 🎯 Focus Session Timer
- Start a **dedicated study session**
- Countdown timer for focused work
- Encourages **deep work using Pomodoro-style sessions**

### 📝 Goal Setting
- Users can define a **study goal before starting a session**
- Helps maintain clarity and productivity

### 📊 Focus Analytics
- Tracks completed focus sessions
- Displays productivity insights and progress

### 🤖 AI Tutor (Study Assistant)
- Chat-based AI tutor section
- Ask study-related questions
- Get instant explanations while studying

### 🔐 User Authentication
- Login / Signup system
- Personalized user experience

### 🎨 Clean & Minimal UI
- Simple and intuitive design
- Distraction-free study environment
